Using vanilla HTML, CSS and jQuery, I created this responsive portfolio website to showcase my abilities, projects, and previous work. You are viewing the result right now!

The "Lil Donut" project was created for a digital class assignment, and in teams we coordinated how to better market our fake brand. The "Lil Donut" brand was created to sell giant sized donuts. As each member was tasked to create a different portion of the marketing scheme, I naturally decided to fork my portfolio website at the time and change it to include a menu, contact page and various other elements. You can view a live version by clicking here!

This project was born as a way to create a link between the Hypixel Minecraft Server's chat and the Discord chat client. It was created back in April of this year, and the core functionality was created in only 7 hours. After that portion was completed, various commands and features were added to ensure it functioned like a real "Discord bot". This is how I expanded my knowledge with NodeJS and JavaScript in general.
